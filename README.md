# Frying4Good

This website was built with Next.js, Discord API, MUI (Material UI), NextAuth, and MongoDB.

## Getting Started

Run npm install to install everything from package.json.

```bash
npm install

```

Then to run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) and you'll be able to see the page appear.
